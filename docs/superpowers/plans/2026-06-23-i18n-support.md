# I18n Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add first-pass `zh-CN` / `en-US` internationalization for the Web workbench, with language selection in Settings.

**Architecture:** Use the Next.js App Router locale segment (`/[locale]`) and a local typed dictionary under `apps/web/src/i18n`. Pass dictionary slices through existing Web components; keep `packages/ui` locale-agnostic and do not touch Worker, Control Plane, DB, OpenAPI, or generated protocol code.

**Tech Stack:** TypeScript, Next.js App Router, React 19, Node built-in test runner, Playwright, pnpm.

## Global Constraints

- Supported locales: `zh-CN` and `en-US`.
- Default locale: `zh-CN`.
- `/` redirects to `/zh-CN`.
- `/zh-CN` renders `<html lang="zh-CN">`; `/en-US` renders `<html lang="en-US">`.
- Settings owns the language selector.
- The selected locale is stored in browser `localStorage`; the root route still redirects to `/zh-CN` in this stage.
- Translate static Web UI copy only: labels, placeholders, button text, accessible labels, titles, empty states, failure text, and page metadata.
- Do not translate user messages, assistant/model messages, markdown content, file paths, command names, diffs, device names, project names, task names, conversation titles, runtime values, API enum values, protocol names, or Worker/API payloads.
- Do not add an i18n dependency, translation platform, automatic translation, server-side preference storage, API changes, Worker changes, Control Plane changes, DB changes, OpenAPI changes, or RTL support.
- Keep changes surgical; do not refactor unrelated layout, datasource, protocol, or package-boundary code.
- Worktree is dirty from other tasks. Stage and commit only the files listed in each task.

- [ ] **Step 3: Add locale helpers**

Create `apps/web/src/i18n/locales.ts`:

```ts
export const supportedLocales = ["zh-CN", "en-US"] as const;

export type Locale = (typeof supportedLocales)[number];

export const defaultLocale: Locale = "zh-CN";

export function isLocale(value: string): value is Locale {
  return supportedLocales.includes(value as Locale);
}

export function normalizeLocale(value: string): Locale {
  return isLocale(value) ? value : defaultLocale;
}
```

```ts
import { NextResponse, type NextRequest } from "next/server";

import { defaultLocale } from "./i18n/locales";

export function proxy(request: NextRequest) {
  const url = request.nextUrl.clone();
  url.pathname = `/${defaultLocale}`;
  return NextResponse.redirect(url);
}

export const config = {
  matcher: "/",
};
```

Create `apps/web/src/app/[locale]/page.tsx`:

```tsx
import { redirect } from "next/navigation";

import { CodexRemoteApp } from "../../components/shell/codex-remote-app";
import { getDictionary } from "../../i18n/dictionary";
import { defaultLocale, isLocale } from "../../i18n/locales";

export default async function Page({ params }: PageProps<"/[locale]">) {
  const { locale: rawLocale } = await params;
  if (!isLocale(rawLocale)) {
    redirect(`/${defaultLocale}`);
  }

  const locale = rawLocale;
  const dictionary = getDictionary(locale);

  return <CodexRemoteApp dictionary={dictionary} locale={locale} />;
}
```

Create `apps/web/src/app/[locale]/layout.tsx`:

```tsx
import type { Metadata } from "next";

import { getDictionary } from "../../i18n/dictionary";
import { normalizeLocale } from "../../i18n/locales";

export async function generateMetadata({ params }: LayoutProps<"/[locale]">): Promise<Metadata> {
  const { locale: rawLocale } = await params;
  const dictionary = getDictionary(normalizeLocale(rawLocale));

  return {
    title: dictionary.metadata.title,
    description: dictionary.metadata.description,
    icons: {
      icon: [{ url: "/codex.svg", type: "image/svg+xml" }],
    },
  };
}

export default async function LocaleLayout({ children, params }: LayoutProps<"/[locale]"> & { children: ReactNode }) {
  const { locale: rawLocale } = await params;
  const locale = normalizeLocale(rawLocale);

  return (
    <html lang={locale}>
      <body>{children}</body>
    </html>
  );
}
```
